# College Management System

A College Management System developed using Python and SQLite for managing student and teacher records, attendance, fees, marks, and reports through a simple desktop application.

## Features

* User Login System
* Password and Security Management
* Student Record Management
* Teacher Record Management
* Attendance Management
* Marks Management
* Fee Management
* Dashboard Overview
* Report Generation
* SQLite Database Integration

## Project Structure

* app.py - Main application file
* Login.py - User authentication module
* Password.py - Password management
* Security.py - Security-related functions
* Dashboard.py - Dashboard interface
* Student.py - Student management module
* Teacher.py - Teacher management module
* Attendance.py - Attendance management
* Marks.py - Marks management
* Fees.py - Fee management
* Report.py - Report generation
* college.db - SQLite database file

## Technologies Used

* Python
* SQLite
* Tkinter (if used for GUI)

## How to Run

streamlit run  "app.py"

## Database

The project uses SQLite database (college.db) to store and manage application data.
