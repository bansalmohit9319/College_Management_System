# College Management System

A College Management System developed using Python, Streamlit, SQLite, and Pandas for managing student records, teacher information, attendance, marks, fees, and reports through a user-friendly dashboard.

---

## Overview

This project provides a centralized platform for managing academic and administrative records within a college environment. It allows users to store, update, view, and manage important institutional data efficiently.

---

## Features

### Student Management
- Add new students
- View student records
- Delete student records
- Store student details including course, email, phone number, and semester

### Teacher Management
- Add teacher records
- View teacher information
- Delete teacher records
- Manage department-wise teacher details

### Attendance Management
- Mark student attendance
- Store attendance date and status
- View attendance records

### Marks Management
- Add student marks
- View marks records
- Delete marks records
- Subject-wise marks tracking

### Fees Management
- Store fee records
- Track paid fees
- Calculate pending fees automatically
- View complete fee details

### Dashboard Analytics
- Total Students
- Total Teachers
- Total Fees Collected
- Recent Student Records

### Reports & Export
- Generate reports for:
  - Students
  - Teachers
  - Attendance
  - Marks
  - Fees
- Export reports in CSV format

### Secure Login System
- User authentication
- Password verification
- Password update functionality
- SHA-256 password hashing for security
